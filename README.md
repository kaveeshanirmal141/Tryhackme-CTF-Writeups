# TryHackMe CTF Writeups

A structured collection of my TryHackMe CTF walkthroughs, notes, and exploitation methodologies.  
This repository is focused on **real offensive security learning**, covering enumeration, exploitation, privilege escalation, and forensic analysis.

---

## Purpose

This repo is built to:
- Document my hands-on cybersecurity learning journey
- Strengthen my penetration testing and red teaming skills
- Build a reference library for future labs and real-world scenarios
- Showcase consistent offensive security practice

---

## What You’ll Find Here

Each writeup typically includes:

- Recon & Enumeration techniques
- Exploitation steps (web, Linux, Windows, Active Directory)
- Privilege Escalation methods
- Flags breakdown and capture process
- Tools used (Nmap, Burp Suite, Gobuster, Netcat, etc.)
- Key lessons learned from each machine
