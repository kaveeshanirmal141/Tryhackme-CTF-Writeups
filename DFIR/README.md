# DFIR CTF Writeups

A curated collection of Digital Forensics & Incident Response (DFIR) CTF writeups, investigations, and analysis notes.

This repository focuses on **realistic incident investigation workflows**, including artifact analysis, timeline reconstruction, malware behavior study, and evidence extraction.

---

## Purpose

This repo exists to:

- Build strong DFIR investigation skills through CTFs and labs  
- Practice structured incident response thinking  
- Learn how attackers behave and how evidence is left behind  
- Develop real-world forensic analysis habits  
- Create a personal DFIR knowledge base for future investigations  

---

## What You’ll Find Here

Each DFIR case study may include:

- Initial incident summary
- System artifacts analysis (Windows/Linux)
- File system investigation
- Memory / process analysis (when applicable)
- Network traffic analysis (PCAPs)
- Timeline reconstruction
- Malware / suspicious activity analysis
- Root cause identification
- Key findings & lessons learned
